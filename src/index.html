USGS Data Viewer

  •  About the application
  •  Disclaimer

The USGS Data Viewer is developed by the Wisconsin Internet Mapping (WiM) group to aid in dissemination of flood sensor data collected in the Short-Term Network database.

It is intended to replace the legacy Hurricane Sandy Storm Tide Mapper that was released in October 2012.

REST map services used in this application:

USGS Short-Term Network (STN) sensor & high-water mark data: stnmapservices.wim.usgs.gov:6080/arcgis/rest/services/STN

NOAA weather radar: gis.srh.noaa.gov/arcgis/rest/services/RIDGERadar/MapServerN

NOAA tropical cyclone track and forecast: nowcoast.noaa.gov/arcgis/rest/services/nowcoast/wwa_meteocean_tropicalcyclones_trackintensityfcsts_time/MapServer

Other useful links:

STN Data Portal (data download): stn.wim.usgs.gov/STNDataPortal/

STN Data Web Services: stn.wim.usgs.gov/STNServices/Documentation.html
